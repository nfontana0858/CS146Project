Calendar Information

Here we will have the main calendar page which will show dates and tasks for each day. Assignments, events, tasks, etc. will all be shown and can be color coded based on given tags/groups. The user has the option to create different calendar tabs based on filters such as groups, tags, color, task type, etc. Reminders of upcoming events may also be shown on this page to ensure the user stays up to date.
